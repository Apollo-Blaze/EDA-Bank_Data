# Exploratory Data Analysis on Bank Stock Market Data

## Overview
This repository contains the code and resources for performing exploratory data analysis (EDA) on bank stock market data. The analysis aims to uncover insights related to the performance of bank stocks, trends in stock prices, and correlations between different financial indicators.

## Objectives
- Analyze the historical stock prices of selected bank stocks.
- Identify trends in stock prices over time.
- Visualize key insights using graphs and charts to aid decision-making.

## Data Source
The dataset used for this analysis is sourced from Yahoo Finance using the `yfinance` library to fetch historical stock market data for various banks.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook
- yfinance

## Getting Started
To run the EDA analysis, follow these steps:

1. Clone the repository:
   ```bash
   git clone https://github.com/Apollo-Blaze/EDA-Bank_Data.git
2. Navigate to the project directory:
   ```bash
   cd bank-stock-eda
3. Install the required packages:
   ```bash
   pip install -r requirements.txt
4. Open the Jupyter Notebook:
   ```bash
   jupyter notebook
5. Run the notebook to see the EDA results and visualizations.

## Results
The EDA results are visualized in the Jupyter Notebook, showcasing key insights about historical stock prices, and trends.

## Conclusion
This exploratory data analysis provides valuable insights into the performance of bank stocks, aiding in strategic decision-making and identifying trends in the stock market.
